# Voces de la extinción - Frontend - Admin

Ésta aplicación está pensada para que los/as administradores de Voces de la Extinción puedan interactuar con la base de datos de la aplicación web.

## Tecnologías

- [React](https://es.react.dev/)
- [React Suite](https://rsuitejs.com/)
- [TanStack Router](https://tanstack.com/router)
- [TanStack Query](https://tanstack.com/query)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vite.dev/)
